# Homepage repo instructions

This repository contains my personal academic homepage.

## General rules

- Preserve the existing site structure, theme, and layout unless explicitly asked.
- Make minimal, targeted edits.
- Do not rewrite pages globally unless requested.
- Do not invent publications, positions, awards, affiliations, links, or dates.
- If information is missing, leave a TODO or ask before adding it.
- Keep tone academic, concise, and professional.
- Prefer clear factual wording over promotional language.
- Do not modify generated build folders unless explicitly needed.

## Content rules

- Homepage text should be short, informative, and academic.
- Research descriptions should emphasize applied economics, urban economics, environmental economics, labor economics, gender inequalities, transport systems, spatial constraints, and policy evaluation where relevant.
- Publications and working papers must match the source files or user-provided text.
- CV, publication, and project links must not be invented.

## Technical rules

- Inspect the site generator before editing.
- Identify whether this is Jekyll, Hugo, Quarto, plain HTML, or another framework.
- Run the appropriate local build or preview command if available.
- Report changed files and any build errors.
